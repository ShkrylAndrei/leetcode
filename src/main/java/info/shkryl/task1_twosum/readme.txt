Решение простое
пробегаемся по каждому элементу массива
и проверяем target-текущий элемент в массиве пристутсвует в мапе,
т.е. формула Число1+Число2=target превращаем
Число2=target-Число1, для поиска второго числа
Если в мапе есть это число, то возращаем индексы обоих чисел
Если числа нет в мапе значит добавляем текущее число в мапу
