Описание решения:
1. пробегаемся по каждому элементу массива
2. проверяем target-текущий элемент в массиве присутствует в мапе,
т.е. формулу Число1+Число2=target, превращаем в формулу:
Число2=target-Число1, для поиска второго числа
3. если в мапе есть это число, то возвращаем индексы обоих чисел
4. если числа нет в мапе, значит добавляем текущее число в мапу и
ищем дальше сумму, пока не найдем
